# Hangman-CLI

>Command Line Interface (CLI) Hangman Game built in NODE.js. To begin, at the command prompt, type `node hm.js`. Good Luck!

### Deployment

1. Clone repo
2. Run `npm install`
3. At command prompt run `node hangman.js`
